Court-File Management System
A web-based application for managing court case files and related documentation in a structured and secure way.
Tech Stack
ASP.NET Core, C#, SQL Server, Entity Framework Core, HTML, CSS 
Project Status- In Progress
Core functionality is implemented, with additional features and refinements planned.
Implemented Features
Court case file creation and management,
Document upload and storage for case-related files,
Basic database structure and data persistence,
Initial user interface for viewing and managing files,
Scheduling files for reviews and note taking
User authentication and role-based access control
Advanced search and filtering for court files
Improved validation and error handling
UI/UX refinements
Deployment and environment configuration
Configuration & Security
Sensitive configuration values (such as connection strings, credentials, and API keys) are not included in this repository.
A sample configuration file (appsettings.example.json) is provided to demonstrate the required settings.
Running the Project Locally
Clone the repository
Create an appsettings.json file based on appsettings.example.json
Update the configuration values as needed
Run the project using Visual Studio or dotnet run
